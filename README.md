# Cafeteria Backend

Este é o backend do projeto da **Cafeteria**, desenvolvido como parte de estudo na faculdade.  

O objetivo é construir uma API simples em PHP, com estrutura organizada e preparada para futuras funcionalidades.

---

## 📁 Estrutura do Projeto

```text
cafeteria-backend/
├── public/                   ⇨ ponto de entrada público (ex: index.php)
├── src/Controllers/          ⇨ controladores da aplicação
├── config/                   ⇨ configurações do sistema
├── .env                      ⇨ variáveis de ambiente (não versionadas)
├── composer.json             ⇨ dependências do projeto
├── .gitignore                ⇨ arquivos/pastas ignorados pelo Git
└── README.md                 ⇨ este arquivo
